## Run Locally

Clone the project

```bash
  git clone https://github.com/ZeroJnr/MounMain.git
```

Go to the project directory

```bash
  cd MounMain
```

Install dependencies

```bash
  cd client && npm i && cd ../server && npm i
```

Start the client

```bash
  npm run start --watch
```

Start the server

```bash
  npm run start:dev
```

## Running Tests

To run tests, run the following command

for the client : 

```bash
  npm run test
```

for the server : 

```bash
  jest run test
```
